Developer API

NBT structure and key classes for developers wanting to integrate with SLMMOEnchant.

NBT Structure

  • MMOITEMS_SLENCHANT — JSON map of enchant ID → level
  • SLENCHANT_MOD_* — stat modifier tags
  • SLENCHANT_PROTECTED — protection stone tag
  • SLENCHANT_SOUL_PVP/PVE — soul counters
  • SLENCHANT_TRAK_* — trak counters

Key Classes (Kotlin)

  • EnchantRegistry — Registry of all enchants
  • EnchantBookBuilder — Create enchant books
  • EnchantDataUtil — Read/write enchant data on items
  • LoreRenderUtil — Render enchant lore
  • StatApplyListener — Apply/remove stat modifiers
  • SetBonusManager — Manage set bonuses
