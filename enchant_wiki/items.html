Items & Systems

Items & Systems

All custom items, souls system, set enchants, trak counters, and enchant slots.

Custom Items

Items created by the plugin, given via /sle giveitem <player> <item_id>.

Lucky DustCONSUMABLE

Drag onto enchant book to increase success rate (+1%/dust, max +15%). Material: GLOWSTONE_DUST

ID: lucky_dust

Slot IncreaserCONSUMABLE

Drag onto item to increase max enchant slots (+1 slot). Material: NETHER_STAR

ID: slot_increaser

Protection StoneCONSUMABLE

Drag onto item to protect once from destruction on enchant failure. Material: TOTEM_OF_UNDYING

ID: protection_stone

Extraction ScrollCONSUMABLE

Required for extraction when require-scroll: true. Held in off-hand during /sle extract. Material: PAPER

ID: scroll

Soul StonePASSIVE

Attach to weapon to start collecting soul points from kills

ID: soul_pvp, soul_pve

Soul BoosterCONSUMABLE

Drag onto weapon with Soul Stone to add bonus soul points

ID: soul_booster_pvp, soul_booster_pve

Trak StonesPASSIVE

Attach to item to count kills/blocks. Three types: Block tracker, Kill tracker, Mob kill tracker

ID: trak_block, trak_kill, trak_mob

Enchant OrbORB

Purchased from Tier Shop. Right-click to reveal a random enchant book of the corresponding tier with randomized success/failure rates

Obtained via: /sle shop

Souls System

Kill mobs/players to increase soul points on a weapon. Soul points affect enchants with soul-cost — each enchant activation deducts soul points.

PlaceholderDescription
%slenchant_soul_pvp%Soul points from player kills
%slenchant_soul_pve%Soul points from mob kills
%slenchant_soul%Total soul points
i

Soul Economy: Use Soul Stones to enable tracking → gain souls from kills → enchants with soul-cost consume souls. Players must manage their soul budget wisely.

Set Enchants

Enchants with bonuses when wearing 2/4/6 items with the same enchant. Like MMO item set bonuses.

KingSET

2pc: +10% ATK DMG | 4pc: +20% ATK DMG + GOLEM_WRATH | 6pc: Full set bonus

WarlordSET

2pc: +5% DEF | 4pc: +15% DEF + HEAVY_CHARGE

PhantomSET

2pc: +10% SPEED | 4pc: +25% SPEED + BLINK

Enchant Slots

Each item has limited enchant slots. Shown on item lore: Enchant: x/max. Use Slot Increaser to add more.

Divine III
Aegis II
 
Enchant: 2/6
i

Base: config.yml → max-enchants-per-item: 6
Extra: Slot Increaser adds +1 per use (up to extra-slot-limit)
Example: Base=6, Extra=3 → Item accepts max 9 enchants.
